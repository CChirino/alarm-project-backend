# alarm-project-backend
[![CircleCI](https://img.shields.io/circleci/project/github/contentful/the-example-app.nodejs.svg)](https://circleci.com/gh/contentful/the-example-app.nodejs) 
The Alarm project is build node.js app teaches the very basics of how to work with Contentful:
-An API was built to bring the alarm system project to life.
## Requirements

* Node 14
* Git
* Mysql
## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/CChirino/alarm-project-backend/
cd alarm-project-backend
```
```bash
npm install
```

```bash
node server.js
```
